# example-skill

Placeholder skill so this repository has at least one discoverable `SKILL.md`.

## When to use

- Verifying `npx skills add maxBRT/skills --list` finds skills
- Copying this folder as a starting point for a new skill

## Instructions

1. Copy `skills/example-skill/` to `skills/<your-skill-name>/`
2. Edit the YAML frontmatter `name` and `description`
3. Replace these instructions with the real agent workflow
4. Commit, push, then install with `npx skills add maxBRT/skills --skill <your-skill-name>`
